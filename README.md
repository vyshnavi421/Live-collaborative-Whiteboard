# Live Collaborative Whiteboard

A real-time collaborative whiteboard built using React, Node.js, Express, Socket.IO, and HTML5 Canvas.

##  Features

-  Real-time collaborative drawing
-  Multiple users drawing simultaneously
-  Color Picker
-  Adjustable Brush Size
-  Clear Whiteboard
-  Download Canvas as PNG
-  Instant synchronization using Socket.IO

## Technologies Used

- React.js
- Node.js
- Express.js
- Socket.IO
- HTML5 Canvas
- CSS3

##  Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
